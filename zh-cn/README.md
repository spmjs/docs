# SPM Handbook

> 本书介绍如何用 [spm](https://github.com/spmjs/spm) 来开发组件和应用。

## 阅读

- [gitbook.io](http://sorrycc.gitbooks.io/spm-handbook/)

## 名词解释

- `源`，详见 [源](registry/README.md)
- `组件`，指可以发布到源上的包，比如 [type 组件](http://spmjs.io/package/type)
